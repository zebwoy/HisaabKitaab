# Madrasah NGO Accounting System

A comprehensive accounting management system for Madrasah NGO (Quranic Studies - Bhiwandi).

## Features

- ✅ Secure password-protected access
- ✅ Income and Expense tracking
- ✅ Category-wise financial breakdown
- ✅ Date range filtering with quick presets
- ✅ Period comparison (current vs previous year)
- ✅ Surplus/Deficit tracking
- ✅ CSV export functionality
- ✅ Detailed financial reports
- ✅ Responsive design

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory, ready to deploy to Netlify or any static hosting service.

## Deployment

This app can be easily deployed to Netlify:

1. Push your code to GitHub
2. Connect your GitHub repo to Netlify
3. Set build command: `npm run build`
4. Set publish directory: `dist`
5. Deploy!

## Usage

1. **First Time Setup**: Set a password when you first access the application
2. **Add Transactions**: Use the "Add Transaction" tab to record income and expenses
3. **View Transactions**: Browse all transactions with date filtering options
4. **Financial Reports**: Access comprehensive reports with:
   - Surplus/Deficit indicators
   - Category-wise breakdowns
   - Period comparisons
   - Exportable data
